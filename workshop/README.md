# Workshop Surface

This directory is reserved for phased, non-runtime work that should not become part of the active deployment surface until it is explicitly promoted.

## Intended contents

- experimental migrations
- prototype clients or adapters
- design drafts
- research notes
- temporary evaluation assets

## Promotion rule

Anything that becomes required for the deployable backend, frontend, governance, or canonical memory export pipeline should move into the active root surface and gain an explicit steward in `governance/agent_roster.yml`.
